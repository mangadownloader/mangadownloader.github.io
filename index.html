Manga Bot

An amazing tool aims to download manga from the world

This tool aims to download manga from the internet to view them at the later time. it will always be 100% FREE. Feel free to feedback to me any errors and new features you want via email dangduy2910@gmail.com or leave comments here. Thanks for using.

Some Features

  • Queue to download multiple manga at the same time
  • Ability to download a manga from an url
  • Automatically create a ZIP or PDF file for each chapter was downloaded completely
  • Advanced search allow to fiend manga by its name or url over multiple sites
  • Automatically update manga list
  • Turn off computer when queue done
